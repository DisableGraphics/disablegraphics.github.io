Installation

Colors uses Meson as its build system internally. It is recommended to have a bit of knowledge about Meson beforehand. However, the Meson team provides newbies with an excellent tutorial

Tutorial

First, create a directory to house the tutorial files. The directory can be named whatever comes to mind.
Then, inside the directoy, create a file named meson.build which contains the following contents:

project('example', 'cpp')
sources = files('main.cpp')
colors_dep = dependency('colors', fallback : ['colors', 'colors_dep'])
executable('example', sources, dependencies : colors_dep)

Following this, create, in the same directory as the meson.build file another file named main.cpp with the following contents:

#include <colors.hpp>
#include <iostream>
int main()
{
std::cout << Colors::compose(Colors::RED, Style::BOLD) << "Hello World!" << Colors::reset() << std::endl;
return 0;
}
Colors::compose
static std::string compose(const Color &fgcolor, const Style &style)
Returns the ANSI escape code for the given color and style.
Colors::reset
static std::string reset()
Resets the command line to the default style and color.

Then, create a subdirectory named subprojects.
Download this file and copy it into subprojects
Now, save and compile the project with the following commands:

meson setup builddir
cd builddir
meson compile
./example

If everything went well, the program should have been executed correctly and in the terminal there should be a "Hello World" message in bold, red letters.
